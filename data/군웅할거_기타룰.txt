무기 특성
가벼움 : 명중 굴림에 +1 추가
기교 : 명중 굴림 시 근력 또는 민첩 보정치 보너스 선택 가능
투척 : 명중 굴림 시 근력 보정치가 1/2로 반감 되지만 적중하면 데미지 1d4 추가
       (다시 주워와야함)
양손 : 양손 착용 필수
다용도 : 양손으로 착용 시 데미지 증가
사거리 : 사거리 증가 뒷칸에 공격 가능 (뒷 칸에 공격 시 명중 굴림 –1)

기본적인 무기 장착 방식 : 오른손과 왼손 장착 가능 *한 턴에 오른손 왼손 둘 다 타격
데미지 방식 : 오른손은 피해 굴림 그대로 데미지 타격, 왼손은 피해 굴림의 1/2
              각각 명중 굴림 실행 

비행동 : 행동을 안 쓰고 사용 가능 (중복x)
한 턴 : 한 턴 유효

기타 용어
인간어 : 인간 사용
드워프어 : 드워프 하플링 노움 사용
엘프어 : 엘프 티블링 사용
오크어 : 오크 사용
드래곤어 : 드래곤본 사용

숙련도 관해
무기 숙련 x -> 명중 굴림할 때 숙련 보너스 x
갑옷 숙련 x -> 장착 불가


기타 공통 : 숙련은 1LV 2, 3LV 3, 5LV 4로 증가한다.
